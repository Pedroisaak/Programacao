No css, existe prioridade de chamadas, caso voce insira um estilo diretamente na tag, ela tem preferencia a um chamado por id e este por classes, atributos e pseudo-classes estes tem prioridade com elementos e pseudo elementos 

        
            .especificidade-css th, td, table{
                border: 1px solid black;
                border-collapse: collapse;
                padding: 10px;
                text-align: center;
            }

No caso acima temos uma especificidade classe e um elemento na conta da tabela abaixo ficaria 0 0 1 1 

        
            #tabela th, #tabela td, #tabela table{
                border: 1px solid red;
                border-collapse: collapse;
                padding: 10px;
                text-align: center;
            }

Neste outro, como chamamos pelo id, a especificidade é maior 0 1 0 1, sendo assim, entre a regra de cima e esta, a configuracao sera aplicada por esta, visto que tem maior especificidade

Podemos usar também o !important para dar maxima prioridade na especificidade, mas não é recomendado
